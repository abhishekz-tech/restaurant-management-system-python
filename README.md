## 📌 Project Overview

This project is a beginner-friendly **Restaurant Management System** built using Python.

The main objective of this project was to practice:
- Python fundamentals
- Logical thinking
- Real-world problem solving
- Menu-driven programming

The system allows users to:
✔️ View restaurant menu  
✔️ Select food items  
✔️ Enter quantity  
✔️ Generate final bill  
✔️ Calculate total amount automatically
